# wordle_solver
An simple algorithm to solve Wordle, currently hosted at https://www.nytimes.com/games/wordle/index.html.

To execute, be sure that you have python3 installed. Open the Wordle page and keep the script running in a Terminal.

# Usage

`python3 wordle.py`

The script will ask for a word and a result provided by Wordle. For example, if you input the word HOUSE into Wordle, and the result is
Green-Yellow-Black-Black-Yellow, you have to input the word HOUSE into the script, and then GYBBY.

The script will then present all the possible words and recommend a random one.
